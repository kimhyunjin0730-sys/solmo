# (주)솔모정보기술 주요 서비스(Service) 상세 안내

## 1. 보안취약점 분석 및 컨설팅 (Security Vulnerability Analysis)
정보자산에 대한 잠재적 위협을 사전에 파악하여 완벽한 보안 대책을 수립합니다.

### **서비스 개요**
* [cite_start]**목적**: 웹/앱 서비스 및 내부 인프라의 취약점을 분석하여 해킹 사고를 미연에 방지 [cite: 9, 174]
* **핵심 활동**: 
    * [cite_start]**모의해킹 (Penetration Testing)**: 외부 전문가 관점에서 실제 해킹 시나리오를 통한 침투 테스트 수행 [cite: 175]
    * [cite_start]**취약점 진단**: 웹 서버, DB, 네트워크 장비 등의 설정 오류 및 보안 약점 점검 [cite: 174]
    * [cite_start]**소스코드 진단 (Secure Coding)**: 개발 단계에서의 취약점 제거를 위한 소스코드 분석 [cite: 175, 211]

### **진단 항목 및 효과**
| 항목 | 내용 | 기대 효과 |
| :--- | :--- | :--- |
| **웹/앱 보안** | SQL Injection, XSS 등 주요 웹 취약점 점검 | 서비스 안전성 확보 및 정보 유출 방지 |
| **인프라 보안** | 서버 OS, DB, 네트워크 장비 설정 최적화 | 인프라 가용성 증대 및 관리 효율화 |
| **컴플라이언스** | ISMS, 개인정보보호법 등 법적 요구사항 준수 점검 | 법적 리스크 해소 및 대외 신뢰도 향상 |

## 2. 보안프린트 / 출력물 보안 (Security Print)
[cite_start]종이 문서를 통한 정보 유출을 원천 차단하는 스마트 보안 출력 솔루션입니다. [cite: 188]

### **서비스 개요**
* [cite_start]**자체 솔루션**: **X-Securitas** 기반의 스크린 및 출력물 워터마크 기술 적용 [cite: 53, 188]
* **주요 기능**:
    * [cite_start]**워터마크 삽입**: 인쇄 시 사용자 정보(ID, IP, 날짜 등)를 투명/반투명하게 삽입하여 유출 경로 추적 [cite: 188]
    * [cite_start]**출력 승인 및 통제**: 민감 정보 포함 시 출력 차단 또는 관리자 승인 후 인쇄 [cite: 161]
    * [cite_start]**이력 관리**: 누가, 언제, 어떤 내용을 출력했는지에 대한 상세 로그 기록 및 감사 [cite: 157, 161]

### **특징**
* 기존 프린터 및 복합기 환경을 그대로 유지하면서 보안 기능만 연동 가능
* 문서 변조 방지 및 출력물에 대한 보안 의식 고취

## 3. 통합 유지보수 서비스 (Total Maintenance)
[cite_start]22년의 노하우와 전문 인력을 바탕으로 고객사의 IT 인프라를 24시간 안정적으로 운영합니다. [cite: 12, 34]

### **서비스 강점**
* [cite_start]**숙련된 기술진**: 총 47명의 인력 중 **특급/고급 기술자 비중이 50% 이상**인 전문 엔지니어 그룹 보유 [cite: 116, 120, 145, 147]
* [cite_start]**맞춤형 서비스**: 단순 장애 처리를 넘어 정기 점검, 성능 최적화, 보안 업데이트 등 선제적 관리 수행 [cite: 160, 172]
* [cite_start]**풍부한 레퍼런스**: 포스코 그룹사, 새마을금고, 금융권 및 공공기관 등 대규모 인프라 운영 경험 [cite: 219]

### **유지보수 범위**
| 구분 | 상세 내용 |
| :--- | :--- |
| **보안 시스템** | [cite_start]UTM, 웹 방화벽(WAF), NAC, DLP 등 보안 솔루션 일체 [cite: 202, 211] |
| **네트워크** | [cite_start]스위치(L2/L3/L4), 무선랜 시스템 관리 및 최적화 [cite: 168, 169, 202] |
| **서버 및 스토리지** | [cite_start]Unix, Linux, Windows 서버 및 데이터 백업 관리 [cite: 169, 202, 211] |
