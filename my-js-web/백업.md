# 하이브리드 환경에 최적화된 통합 데이터 보호 솔루션

하이브리드 IT 인프라 기업 환경에 적합한 **데이터 보호 및 시스템 백업·복구 솔루션**입니다. 사용자 친화적인 웹 기반 관리 콘솔을 통해 다양한 플랫폼에서 안정된 백업과 빠른 복구를 제공하여 기업의 자산을 안전하게 지킵니다.

---

## 🛠️ 주요 기능

| 기능 항목 | 상세 설명 |
| :--- | :--- |
| **이미징 백업** | 시스템 전체 및 특정 파일 단위의 정밀한 이미징 백업 수행 |
| **재해 복구** | 사고 발생 시 즉각적인 복구 및 이종 하드웨어로의 마이그레이션 지원 |
| **중복 제거** | 데이터 중복 제거 기술을 통해 저장 공간 효율 극대화 |
| **데이터 보안** | 강력한 암호화 및 블록체인 기반의 데이터 공증(Notary) 지원 |

---

## 💡 도입 기대 효과

* **비즈니스 연속성 확보**: 시스템 장애 시 즉각적인 복구를 통해 업무 중단 시간을 최소화합니다.
* **비용 절감**: 데이터 중복 제거 기능을 활용하여 백업용 스토리지 구매 비용을 절감합니다.
* **능동적 위협 예방**: 백업 데이터에 대한 랜섬웨어 공격 등을 실시간으로 감지하고 예방합니다.
* **데이터 무결성 보장**: 블록체인 방식의 공증 기술을 사용하여 백업 데이터의 위변조 여부를 완벽히 확인합니다.

---

## 🏗️ 시스템 구성 및 플랫폼 지원

### 지원 플랫폼 (21개 이상의 환경 지원)
* **물리 시스템**: Windows, Linux 서버 및 워크스테이션
* **가상화**: VMware, Hyper-V, Citrix 등 주요 하이퍼바이저
* **클라우드 및 앱**: Microsoft 365, Azure, Amazon Web Services(AWS), Google Cloud 등

### 구성 방식
1. **On-Premise 구성**: 사내에 관리 서버와 스토리지(SAN, TAPE 등)를 직접 구축하여 독자적으로 운영
2. **Cloud 구성**: 아크로니스 데이터 센터(Acronis Data Center)를 활용하여 별도의 인프라 구축 없이 클라우드 기반의 백업 및 관리 수행
