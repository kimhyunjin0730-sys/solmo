# 자동화된 패치관리시스템(PMS) 구축을 통한 전사적인 보안 수준 강화

윈도우 계열의 **PC 및 서버에 대한 보안 강화**를 위해 패치관리시스템을 구축합니다. 운영체제의 신속한 보안 패치 및 백신, PC보안, 개인정보관리 등 필수 소프트웨어의 **중앙 관리 체계**를 확립하여 보안 공백을 최소화하고 정보 자원의 안정성을 확보합니다.

---

## 🛠️ 주요 기능

| 기능 항목 | 상세 내용 |
| :--- | :--- |
| **자동 패치 관리** | 운영체제 및 어플리케이션에 대한 패치 자동/강제 설치 및 업데이트 수행 |
| **현황 관리** | 전사 단말의 패치 적용 현황 수집 및 상세 통계 리포트 제공 |
| **취약점 제거** | 자동화된 패치 관리를 통해 보안 약점을 사전에 제거하고 수준 강화 |
| **단말 보호** | 해킹 및 자료 유출 방지 등 PC 보호를 통한 보안 사고 최소화 |
| **가용성 확보** | 악성코드 및 좀비 PC로 인한 불필요 트래픽 발생 방지 및 네트워크 안정성 유지 |

* **위협 관리**: 악성코드 유포 및 좀비 PC화를 방지하여 전사 네트워크 보안을 강화합니다.

# 스마트한 문서 관리의 시작, Gaaiho PDF

Gaaiho는 PDF 리더, 편집기, 변환기 및 관리 도구를 포함하는 통합 솔루션으로, 기업의 문서 업무 프로세스를 혁신합니다. 단순한 뷰어를 넘어 강력한 편집 기능과 스마트한 관리 기능을 제공합니다.

---

## 🛠️ 주요 구성 어플리케이션

| 어플리케이션 | 상세 기능 및 특징 |
| :--- | :--- |
| **1. Gaaiho PDF DOC** | PDF 문서 편집 등 대부분의 전문적인 기능을 구현하는 메인 어플리케이션 프로그램입니다. |
| **2. Gaaiho PDF Converter** | 다양한 생성 기능(병합, 분할, 대량변환)과 함께 PDF를 워드나 엑셀 형식으로 바꾸는 역변환 기능을 제공합니다. |
| **3. Gaaiho PDF Manager** | 내 PC에 흩어져 있는 수많은 PDF 문서들을 손쉽게 탐색하고 효율적으로 관리할 수 있도록 돕습니다. |

---

## ✨ 핵심 기능 (Key Features)

* **강력한 편집 및 뷰어**: 단순 리더 기능을 넘어 텍스트와 이미지를 직접 수정하는 강력한 편집 환경을 제공합니다.
* **다양한 PDF 생성 및 변환**: 대량의 파일을 한꺼번에 변환하거나 여러 문서를 하나로 병합하고 분할하는 작업이 가능합니다.
* **PDF OCR (광학문자인식)**: 이미지 형태의 PDF를 검색 및 편집이 가능한 텍스트 데이터로 변환합니다.
* **스마트 관리 도구**: 편리한 PDF 관리(Manager) 기능을 통해 업무 생산성을 높여줍니다.
* **풍부한 부가 기능**:
    * **클라우드 연결**: 어디서나 문서에 접근하고 저장할 수 있는 클라우드 연동 지원.
    * **디지털 서명**: 종이 없는 업무 환경을 위한 전자 서명 기능 제공.
    * **파일 최적화**: 고품질을 유지하면서 파일 사이즈를 축소하는 기능 및 Office 메뉴 연동.
    * **주석 및 소통**: 다양한 주석 도구와 파일 음성 읽기 기능으로 협업 효율 증대.

---

## 📈 도입 기대 효과

1. **업무 효율성 증대**: 워드나 엑셀로의 역변환 기능을 통해 문서 재작성 시간을 획기적으로 줄여줍니다.
2. **비용 절감**: 강력한 PDF 편집 기능으로 별도의 고가 소프트웨어 없이도 전문적인 문서 작업이 가능합니다.
3. **디지털 전환(DX) 가속**: 디지털 서명과 클라우드 연동을 통해 페이퍼리스(Paperless) 사무 환경을 구축합니다.
