# 위협 관리의 통합 (Unified Threat Management)

Firewall, VPN, Anti-Virus, IDS, IPS, Anti-Spam, SSL VPN, QoS 등의 여러 보안 기능을 **단일의 장비에서 통합적으로 수행**하는 시스템입니다.

---

## 🚀 주요 보안 기능 요약

| 구분 | 핵심 내용 |
| :--- | :--- |
| **차세대 보안** | ASIC-accelerated 기반의 하드웨어 및 전용 Secure OS 탑재 |
| **침입 차단** | 5,000종 이상의 공격 패턴 실시간 차단 및 자동 업데이트 |
| **안티바이러스** | 네트워크 유입 악성 파일 실시간 제거 및 성능 최적화 |
| **익명성/VPN** | 다양한 암호화 프로토콜 지원 및 웹 브라우저 접속(Web mode) 제공 |

---

## 🔍 세부 핵심 기능

### ### **1. 차세대 보안의 기준**
* **하드웨어 가속:** ASIC-accelerated 기반의 하드웨어와 전용 Secure OS 사용
* **모드 지원:** NAT, PAT, Transparent(bridge) mode 지원
* **가상화:** Virtual Domain 기능 제공 (NAT/Transparent mode)
* **네트워크 편의:** DHCP relay, WINS 지원

### ### **2. 능동적인 침입 차단**
* **강력한 방어:** 5,000 종류 이상의 공격 패턴 차단
* **자동 업데이트:** 실시간 침입 차단 패턴 자동 업데이트 시스템
* **유연한 정책:** 서비스별 또는 IP별로 개별 정책 적용 가능
* **특수 공격 대응:** DOS/DDOS, SCAN, Flood, ICMP 공격 차단

### ### **3. 안티바이러스의 모든 것**
* **사전 방역:** 네트워크로 들어오는 악성 파일로부터의 위협을 원천 방지
* **FortiResponse Network:** 최신 위협에 대해 지속적인 자동 업데이트 제공
* **성능 최적화:** 악성 코드가 내부 네트워크에 들어오기 전, 속도 저하 없이 제거

### ### **4. 철저한 익명성 및 VPN**
* **다양한 프로토콜:** PPTP, IPSec, SSL, L2TP 지원
* **강력한 암호화:** DES, 3DES, AES 암호화 지원
* **인증 옵션:** SHA-1, MD5 인증 옵션 제공
* **접근성:** 간단하게 웹 브라우저로도 사용 가능 (web mode)

* **통합 관리**: 분산된 네트워크 환경에서도 지사별 설치 및 중앙 통합 관리를 지원

# 네트워크 접근 제어 (NAC: Network Access Control)

강력한 인증을 통해 **자산과 사용자를 식별**하고, 네트워크 접근에 대한 권한을 차등 부여합니다. 특정 단말 및 IP/서브넷/VLAN 등 사용자 접속 권한을 제어하며, 단말의 보안 상태를 점검 및 조치하여 내부 보안 관리 체계를 완성합니다.

---

## 🛠️ 주요 기능

| 기능 항목 | 상세 설명 |
| :--- | :--- |
| **네트워크 실명화** | 네트워크를 사용 중인 단말의 인증을 통한 사용자 식별 및 관리 |
| **외부 사용자 통제** | 방문자 및 외부 협력 업체의 네트워크 사용 범위를 제한 |
| **단말 무결성 관리** | 보안 정책 준수 여부에 따른 단말 관리 및 PC 무결성 검사 수행 |
| **접근 권한 분리** | 사용자 직급이나 역할 등 권한에 따른 네트워크 접근 구역 분리 |

---

## 💡 도입 기대 효과

* **보안 정책 집중화**: 내부 네트워크를 사용하는 모든 단말에 대해 일관된 보안 정책 적용
* **규정 이행 강제화**: 사내 보안 규정을 지키지 않은 단말의 접속을 차단하여 규정 준수 유도
* **내부 보안성 강화**: 보안 미준수 단말을 네트워크에서 즉시 분리하여 내부 위협 확산 방지
* **청정한 네트워크 유지**: BYOD(개인 기기 업무 활용) 환경에서도 유무선을 아우르는 통합 관리 가능

---

## 🏗️ 시스템 구성 및 플랫폼

### 서비스 제공 형태
* **On-Premise (구축형)**: 기관 및 기업의 독자적인 운영이 가능한 형태로, 국내 환경에서 가장 선호되는 구축 방식입니다.
* **CLOUD (클라우드)**: 중소기업 또는 MSP 사업자를 위한 구성 형태로 초기 인프라 부담이 적습니다.
* **VM (가상 머신)**: MSP, MSSP, CSP 등 다양한 플랫폼 환경을 지원하며 가상화 환경에 최적화되어 있습니다.

### 주요 구성 요소
1. **Policy Server & Console (정책서버 & 콘솔)**: 유무선 네트워크를 통합 관리하고 보안 정책을 배포합니다.
2. **Network Sensor (차단센서)**: 네트워크 접점에서 단말 정보를 수집하고 정책에 따른 강력한 접속 통제를 수행합니다.
3. **Agent (에이전트)**: PC 자산 관리 및 장치 사용 통제를 수행하며, 필요에 따라 선택적으로 설치하여 사용할 수 있습니다.

# 통합계정관리 및 접근통제 (HIWARE)

주요 정보시스템의 **계정 생성부터 수정, 삭제, 권한 회수까지의 전 과정(Life-cycle)을 관리**합니다. 또한 장비에 접속하는 모든 사용자의 권한과 명령어를 통제하며, 이에 대한 상세 감사를 통해 **컴플라이언스 준수 및 보안 감사**에 완벽하게 대응할 수 있는 통합 솔루션입니다.

---

## 🛠️ 주요 기능

| 기능 항목 | 상세 설명 |
| :--- | :--- |
| **접근 차단** | 인증 등록이 되지 않은 IP 및 MAC 주소로부터의 모든 접근을 원천 차단 |
| **중앙 집중 통제** | 접속 IP, MAC, 접속 가능 시간 설정 및 2-Factor 인증을 통한 권한 분배 |
| **명령어 제어** | 그룹 별, 장비 별 등 다양한 조건에 따른 명령어 권한 설정 및 실시간 통제 |
| **스케줄 관리** | 작업 시간 및 유휴 시간을 날짜, 요일, 시간 단위로 세밀하게 설정 가능 |
| **실시간 모니터링** | 현재 작업 중인 모든 세션에 대해 실시간 모니터링 및 감사 수행 |

---

## 💡 도입 기대 효과

* **접속 경로 일원화**: 운영 시스템에 접속하는 모든 사용자의 경로를 하나로 묶어 관리 효율성 증대
* **보안 체계 강화**: 주기적인 패스워드 변경 강제화 및 컴플라이언스 준수를 통해 보안성 극대화
* **중앙 관리 체계**: 모든 운영 시스템의 계정 정보와 보안 정책을 중앙에서 한눈에 관리 및 제어

---

## 🏗️ 시스템 구성 특징

### 통합 관리 아키텍처
* **이중화 구성**: 서비스의 연속성을 보장하기 위해 통합계정관리시스템을 이중화(High Availability)로 구축합니다.
* **시스템 연동**: 기업 내 기존 **HR(인사) 시스템 및 전자결재 시스템**과의 유연한 연동 개발을 지원합니다.
* **대상 장비 지원**: Linux, Unix, Windows Server 등 기업 내 다양한 대상 장비와 연동되어 통합 관리를 수행합니다.
