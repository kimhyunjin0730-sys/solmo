### 회사 소개

(주)솔모정보기술은 보안솔루션 구축 및 유지보수와 소프트웨어, 보안시스템 납품 등 기업의 전산실 구축 및 수탁 보안 운영을 수행하고 있습니다. 또한 경쟁력 있는 솔루션을 통해 고객사의 가치와 만족도를 높이도록 노력하며, 고객과 함께하는 기업이 되도록 최선을 다하고 있습니다.

SI, NI 구축과 전문 유지보수, 위탁운영, 전문보안 솔루션구축, 전문 컨설팅 등의 **IT토탈 서비스 사업**을 제공합니다.

Firewall, VPN, Anti-Virus, IDS, IPS, Anti-Spam, SSL VPN, QoS 등 여러 보안 기능을 **단일의 장비에서 통합적으로 수행**하는 차세대 보안 시스템입니다.

#### 주요 보안 기능

| 구분 | 핵심 내용 |
| :--- | :--- |
| **차세대 보안** | ASIC 기반 가속화 하드웨어 및 전용 Secure OS 탑재 |
| **침입 차단** | 5,000종 이상의 공격 패턴 실시간 차단 |
| **안티바이러스** | 네트워크 유입 악성 파일 실시간 제거 및 업데이트 |
| **익명성/VPN** | 다양한 암호화 프로토콜 지원 및 웹 모드 제공 |

#### 세부 핵심 기능

**1. 차세대 보안의 기준**
- **하드웨어 가속:** ASIC-accelerated 기반의 하드웨어와 전용 Secure OS 사용
- **모드 지원:** NAT, PAT, Transparent(bridge) mode 지원
- **가상화:** Virtual Domain 기능 제공 (NAT/Transparent mode)
- **네트워크 편의:** DHCP relay, WINS 지원

**2. 능동적인 침입 차단**
- **강력한 방어:** 5,000 종류 이상의 공격 패턴 차단
- **자동 업데이트:** 실시간 침입 차단 패턴 자동 업데이트 시스템
- **유연한 정책:** 서비스별 또는 IP별로 개별 정책 적용 가능
- **특수 공격 대응:** DOS/DDOS, SCAN, Flood, ICMP 공격 차단

**3. 안티바이러스**
- **사전 방역:** 네트워크로 들어오는 악성 파일로부터의 위협을 원천 방지
- **FortiResponse:** 최신 위협에 대해 지속적인 자동 업데이트 제공
- **성능 최적화:** 악성 코드가 내부 네트워크에 들어오기 전, 속도 저하 없이 제거

**4. VPN / 익명성**
- **다양한 프로토콜:** PPTP, IPSec, SSL, L2TP 지원
- **강력한 암호화:** DES, 3DES, AES 암호화 지원
- **인증 옵션:** SHA-1, MD5 인증 옵션 제공
- **접근성:** 간단하게 웹 브라우저로도 사용 가능 (web mode)

**카테고리:** 네트워크 보안 | **제조사:** PIOLINK

웹 서버에 존재하는 **보안 취약점과 웹 공격을 탐지**하고 이를 차단하기 위해 도입된, 웹 어플리케이션 보안만을 위한 특화 솔루션입니다.

| **변조 차단** | 웹 사이트 내용이 임의로 변조되는 것을 사전에 차단 |

#### 도입 기대 효과

- **웹 서비스 보호:** 외부 공격으로부터 핵심 웹 서비스를 안전하게 보호
- **손실 방지:** 웹 공격으로 인한 서비스 중단 및 비즈니스 손해를 제거

- **법적 근거 확보:** 보안 위반 사례 발생 시 법적 대응을 위한 근거 확보
- **고객 신뢰 향상:** 안정적인 서비스 운영을 통해 고객의 신뢰 확보

#### 시스템 구성 및 특징

- **부하분산 (Load Balancing):** 서버로 유입되는 트래픽을 효율적으로 분산 처리
- **SSL 오프로딩:** 암호화 트래픽 처리를 전담하여 웹 서버의 부하 경감

### 7-4. 유해사이트 차단 솔루션 (WebkeeperSG™)

임직원의 인터넷 사용에 대한 통제 및 관리를 통해 **비즈니스 정보의 외부 유출을 방지**합니다. 악성코드 유포 및 파일공유 사이트를 차단하여 내부 사용자의 중요 데이터를 보호하고 **시스템의 안정성을 창출**하는 보안 서비스입니다.

| **HTTPS(SSL) 통제** | 암호화된 HTTPS 접속에 대해서도 정교한 접속 통제 및 관리 가능 |
| **트래픽 및 대역폭 제어** | P2P, 웹하드 등 네트워크 대역폭을 많이 점유하는 프로그램 통제 |

- **보안 위협 근원적 차단:** 악성코드 유포의 주요 경로인 P2P 및 웹하드 차단으로 DDoS 공격 등을 사전에 방지

- **배치:** 본사 및 지사의 백본 스위치(Backbone Switch)에 구성
- **통합 관리:** 분산된 네트워크 환경에서도 지사별 설치 및 중앙 통합 관리 지원

### 7-5. Genian NAC (네트워크 접근 제어)

**카테고리:** 네트워크 보안 | **제조사:** Genian

강력한 인증을 통해 **자산과 사용자를 식별**하고, 네트워크 접근에 대한 권한을 차등 부여합니다. 특정 단말 및 IP/서브넷/VLAN 등 사용자 접속 권한을 제어하며, 단말의 보안 상태를 점검 및 조치하여 내부 보안 관리 체계를 완성합니다.

#### 주요 기능

| 기능 항목 | 상세 설명 |
| :--- | :--- |
| **네트워크 실명화** | 네트워크를 사용 중인 단말의 인증을 통한 사용자 식별 및 관리 |
| **외부 사용자 통제** | 방문자 및 외부 협력 업체의 네트워크 사용 범위를 제한 |
| **단말 무결성 관리** | 보안 정책 준수 여부에 따른 단말 관리 및 PC 무결성 검사 수행 |
| **접근 권한 분리** | 사용자 직급이나 역할 등 권한에 따른 네트워크 접근 구역 분리 |

#### 도입 기대 효과

- **보안 정책 집중화:** 내부 네트워크를 사용하는 모든 단말에 대해 일관된 보안 정책 적용
- **규정 이행 강제화:** 사내 보안 규정을 지키지 않은 단말의 접속을 차단하여 규정 준수 유도
- **내부 보안성 강화:** 보안 미준수 단말을 네트워크에서 즉시 분리하여 내부 위협 확산 방지
- **청정한 네트워크 유지:** BYOD 환경에서도 유무선을 아우르는 통합 관리 가능

#### 시스템 구성 및 플랫폼

- **On-Premise:** 기관 및 기업의 독자적인 운영이 가능한 구축형 (국내 환경에서 가장 선호)
- **CLOUD:** 중소기업 또는 MSP 사업자를 위한 구성 형태로 초기 인프라 부담이 적음
- **VM:** MSP, MSSP, CSP 등 다양한 플랫폼 환경을 지원하며 가상화 환경에 최적화

주요 구성 요소:
1. **Policy Server & Console (정책서버 & 콘솔):** 유무선 네트워크를 통합 관리하고 보안 정책을 배포
2. **Network Sensor (차단센서):** 네트워크 접점에서 단말 정보를 수집하고 정책에 따른 강력한 접속 통제 수행
3. **Agent (에이전트):** PC 자산 관리 및 장치 사용 통제 수행, 필요에 따라 선택적 설치 가능
